A simple book recommendation app that suggests titles similar to your favorite books using collaborative filtering. Built with Streamlit and KNN, it analyzes user rating patterns from the Book-Crossing dataset to find books with similar appeal. Just type a book title, and get instant recommendations with similarity scores.
